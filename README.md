# NIRD - Le Village Résistant Numérique

Application web interactive pour promouvoir la démarche NIRD (Numérique Inclusif, Responsable et Durable).

## Objectif

Cette application aide les établissements scolaires, élèves, enseignants, familles et collectivités à comprendre comment réduire leurs dépendances numériques et adopter progressivement la démarche NIRD.

##  Fonctionnali

- **Parcours interactif** : Découvrez NIRD selon votre profil (élève, enseignant, collectivité, famille)
- **Simulateur d'économies** : Calculez les économies réalisées en adoptant NIRD
- **Guides et ressources** : Accès aux guides de migration, tutoriels et formations
- **Communauté** : Rejoignez la communauté NIRD et partagez vos expériences
- **Gamification** : Badges, défis et quiz pour rendre l'apprentissage ludique


##  Technologies

- HTML5
- CSS3 (avec animations et transitions)
- JavaScript vanilla (pas de dépendances)
- Fonts Google (Fredoka, Inter) - libres d'utilisation
- Architecture modulaire : composants séparés chargés dynamiquement
